# NLP Sentiment Analysis

This project implements a Natural Language Processing (NLP) based sentiment analysis model that classifies customer reviews as positive or negative.

Sentiment analysis is widely used by businesses to understand customer feedback, analyze product reviews, and improve decision making.

---

## Project Overview

The goal of this project is to build a machine learning model capable of identifying sentiment in text data.

Using NLP preprocessing techniques and machine learning algorithms, the model predicts whether a review expresses positive or negative sentiment.

---

## Features

• Text preprocessing and cleaning  
• NLP feature extraction using TF-IDF  
• Machine learning model training  
• Sentiment prediction for customer reviews  

---

## Tech Stack

Python  
Pandas  
Scikit-learn  
NLTK  

---

## Dataset

The dataset contains customer product reviews and sentiment labels.

Columns include:

Review_Text  
Sentiment

Sentiment values:

Positive  
Negative

---

## Machine Learning Approach

1. Data cleaning
2. Text preprocessing
3. Tokenization
4. TF-IDF feature extraction
5. Model training using Logistic Regression
6. Sentiment prediction

---

## Example

Input Review:

"This product is amazing and works perfectly."

Predicted Sentiment:

Positive
